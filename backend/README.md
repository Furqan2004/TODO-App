# CLI Todo App (Phase I)

A simple command-line todo application that stores all tasks in memory during the session.

## Running the Application

Requires Python 3.13+ and `uv`.

```bash
cd backend
uv run todo
```

## Testing

```bash
cd backend
uv run pytest
```

## Features

- Add tasks
- List all tasks
- Update tasks
- Delete tasks
- Toggle task status between pending and complete
